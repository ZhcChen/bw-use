# 项目协作规则

## 测试数据保护

- 开发、调试、联调、验收测试时，**禁止修改、删除、覆盖用户现有正式数据**。
- 现有正式数据包括但不限于：`data/` 下已有数据库、已有 profile、已有浏览器实例记录、用户手动创建的数据文件。
- 需要做联调或端到端验证时，必须使用**隔离的临时数据目录/临时数据库/临时端口/临时进程**，例如通过 `BW_USE_DATA_DIR` 指向独立临时目录。
- 测试过程中新增的临时数据、临时目录、临时浏览器实例、日志、截图、快照、缓存，**在测试完成后必须立即清理**。
- 除非用户明确授权，否则对现有正式数据只允许只读检查，不允许写入操作。
